# Innuendo por El Crucero del Amor

Landing premium para una experiencia privada en embarcacion por el Lago Calafquen, Lican Ray, Chile.

## Desarrollo

```bash
npm install
npm run dev
```

## Estructura inicial

- `app/`: paginas principales con Next.js App Router.
- `components/layout/`: header, footer y acceso flotante a WhatsApp.
- `components/sections/`: secciones visuales de la landing.
- `components/ui/`: componentes reutilizables.
- `data/`: contenido editable de la experiencia.
- `lib/`: constantes y helpers.
- `public/images/innuendo/`: carpeta para fotografias reales.
- `styles/`: tokens visuales premium.
